# 🛒 Node Shopee Cart

Simulação de um carrinho de compras inspirada na Shopee, desenvolvida com Node.js e JavaScript. O projeto executa diretamente no terminal e permite adicionar, remover, listar produtos e calcular o valor total da compra.

## 🚀 Funcionalidades

* ✅ Adicionar produtos ao carrinho
* ✅ Listar produtos do carrinho
* ✅ Remover uma unidade de um produto
* ✅ Excluir um produto do carrinho
* ✅ Calcular subtotal dos itens
* ✅ Calcular o valor total da compra

## ▶️ Como Executar

### Pré-requisitos

* Node.js instalado

### Clone o repositório

```bash
git clone https://github.com/DiegoSantiago1/node-shopee-cart.git
```

### Entre na pasta do projeto

```bash
cd node-shopee-cart
```

### Execute a aplicação

```bash
node src/index.js
```
