# 🛒 Node Shopee Cart

Simulação de um carrinho de compras inspirada na Shopee, desenvolvida com Node.js e JavaScript. O projeto executa diretamente no terminal e permite gerenciar produtos dentro de um carrinho de compras através de operações como adicionar, remover, listar itens e calcular o valor total da compra.

## 🚀 Funcionalidades

* Adicionar produtos ao carrinho
* Listar produtos do carrinho
* Remover uma unidade de um produto
* Excluir um produto do carrinho
* Calcular subtotal dos itens
* Calcular o valor total da compra

## 🛠️ Tecnologias Utilizadas

* JavaScript (ES Modules)
* Node.js
* Git
* GitHub

```text
src/

```

## ▶️ Como Executar

Clone o repositório:

```bash
git clone https://github.com/SEU-USUARIO/node-shopee-cart.git
```

Acesse a pasta do projeto:

```bash
cd node-shopee-cart
```

Execute a aplicação:

```bash
node src/index.js
```
